# Backend Controllers

Este directorio contiene todos los controladores de la API.

Cada controlador maneja la lógica de negocio para una entidad específica:
- authController.js → Autenticación y registro
- usuariosController.js → Gestión de usuarios
- sucursalesController.js → Gestión de sucursales
- empleadosController.js → Gestión de empleados
- clientesController.js → Gestión de clientes
- serviciosController.js → Gestión de servicios
- citasController.js → Gestión de citas
- pagosController.js → Gestión de pagos y comisiones
- reportesController.js → Generación de reportes
