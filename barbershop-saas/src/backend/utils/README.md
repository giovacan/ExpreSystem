# Backend Utils

Este directorio contiene funciones utilitarias reutilizables.

Utilidades incluidas:
- jwt.js → Generación y validación de tokens JWT
- hash.js → Hashing y verificación de contraseñas
- validators.js → Validadores reutilizables (email, teléfono, etc.)
- activityLog.js → Registro de auditoría
- dateUtils.js → Utilidades para fechas
- errorUtils.js → Construcción de errores estandarizados
- response.js → Construcción de respuestas estandarizadas
