# Frontend Services

Este directorio contiene servicios para comunicación con la API.

Servicios incluidos:
- authService.js → Autenticación, login, registro
- usuariosService.js → CRUD de usuarios
- sucursalesService.js → CRUD de sucursales
- empleadosService.js → CRUD de empleados
- clientesService.js → CRUD de clientes
- serviciosService.js → CRUD de servicios
- citasService.js → CRUD de citas
- pagosService.js → Pagos y comisiones
- reportesService.js → Obtención de reportes
- apiClient.js → Cliente HTTP base (axios configurado)

Los servicios usan axios para hacer peticiones HTTP al backend.
