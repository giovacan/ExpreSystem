# Frontend Components

Este directorio contiene componentes reutilizables de React.

Componentes principales:
- Layout.jsx → Layout general (header, sidebar, footer)
- Login.jsx → Componente de login
- Dashboard.jsx → Dashboard principal
- Agenda.jsx → Vista de agenda/calendario
- Clientes.jsx → CRUD de clientes
- Servicios.jsx → CRUD de servicios
- Empleados.jsx → CRUD de empleados
- Pagos.jsx → Formulario de registro de pagos
- Reportes.jsx → Vistas de reportes
- Forms/ → Componentes de formularios reutilizables
- Tables/ → Componentes de tablas reutilizables
- Modals/ → Componentes de modales

Todos los componentes usan TailwindCSS para estilos.
