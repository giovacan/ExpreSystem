# Frontend Hooks

Este directorio contiene custom hooks reutilizables.

Hooks incluidos:
- useAuth.js → Hook para gestionar autenticación
- useFetch.js → Hook para peticiones HTTP
- useForm.js → Hook para gestión de formularios
- usePermisos.js → Hook para validar permisos del usuario
- useLocalStorage.js → Hook para persistencia en localStorage
- usePagination.js → Hook para paginación
- useModal.js → Hook para gestión de modales
- useNotification.js → Hook para notificaciones

Los hooks encapsulan lógica reutilizable y mejoran la legibilidad del código.
