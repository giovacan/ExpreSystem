# Frontend Pages

Este directorio contiene las páginas principales de la aplicación.

Páginas incluidas:
- LoginPage.jsx → Página de autenticación
- DashboardPage.jsx → Dashboard principal
- AgendaPage.jsx → Página de agenda
- ClientesPage.jsx → Página de gestión de clientes
- ServiciosPage.jsx → Página de gestión de servicios
- EmpleadosPage.jsx → Página de gestión de empleados
- PagosPage.jsx → Página de pagos y comisiones
- ReportesPage.jsx → Página de reportes
- SettingsPage.jsx → Página de configuración
- ProfilePage.jsx → Perfil del usuario

Cada página corresponde a una ruta en React Router.
