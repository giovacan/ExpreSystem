# Frontend Styles

Este directorio contiene los estilos globales de la aplicación.

Archivos incluidos:
- globals.css → Estilos globales y reset
- tailwind.config.js → Configuración de TailwindCSS
- variables.css → Variables CSS personalizadas (colores, espaciado, etc.)
- responsive.css → Estilos responsivos personalizados

La mayoría de los estilos se usan directamente con clases de TailwindCSS.
Los estilos CSS personalizados se utilizan solo cuando es necesario ir más allá de TailwindCSS.
